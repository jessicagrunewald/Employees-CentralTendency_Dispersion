### Medidas de tendência central e dispersão

Atividade básica de estatística aplicada com medidas de tendência central e dispersão, aplicadas em uma pequena base de dados, de funcionários fictícios, em formato .csv.

Essa atividade foi realizada para a disciplina de Estatística Aplicada do curso de Especialização em Big Data, Data Science and Data Analytics pela Universidade Unisinos.

Linguagem de programação: R
</br>
IDE: RStudio
